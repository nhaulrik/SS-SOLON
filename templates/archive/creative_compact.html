Key Epics & Features

Hexagonal Architecture Refactor
Restructure revenue-management-be-custom to clean hexagonal architecture with improved separation of concerns, enhanced testability, and reduced coupling across domain boundaries.
Java 25 Upgrade
Migrate codebase to Java 25 to leverage latest language features, virtual threads, pattern matching, and significant performance improvements for tax operations.
Spring Boot 4.x Migration
Upgrade Spring Boot framework to latest version with enhanced observability, improved dependency management, native compilation support, and critical security patches.
Oracle 26ai Database
Migrate to Oracle 26ai to harness AI-powered optimization, advanced analytics, and improved query performance for high-volume tax transaction processing.
Tech Debt Across Teams
Systematic UI improvements and technical debt reduction initiatives spanning ATH, EOS, SPA, HEL, and DAID teams for long-term platform sustainability.
Config Items Management
Build dedicated tool for safely packaging, versioning, and migrating configurations across development, staging, and production environments without manual intervention.
