=== DATA EXTRACTION RULES (read before filling any field) ===

PI FILTER SCOPE:
- The raw feature rows you receive have already been pre-filtered to the active PI.
- The shared reference sheet (Initiative Stats) is NOT pre-filtered.
- For every lookup in Initiative Stats, you MUST match the PI column exactly.
- Never read a value from a row whose PI column does not match the active PI.

LOOKUP DISCIPLINE — NO CALCULATION:
- Total Estimate and Feature Count are direct cell reads from Initiative Stats.
- Do NOT sum, count, or derive these values yourself.
- If you cannot find the matching row, write N/A — never substitute a calculated value.
- Feature-level data (name, description, status, estimate, requester, IRM) is read from raw feature rows only.

SOURCE PRIORITY:
- Initiative Stats sheet → aggregate numbers (Total Estimate, Feature Count, P1/P2/P3)
- Raw feature rows      → all per-feature fields (name, description, status, estimate, requester, IRM)
- Never cross-use these sources (e.g. do not derive feature counts from raw rows when the value is in Initiative Stats).

BAR PERCENTAGE RULE:
- Identify the feature with the highest Estimate value among all raw feature rows for this initiative (ignoring blank/TBD).
- That feature receives --bar-pct: 100%.
- All other features: (feature_estimate ÷ max_estimate) × 100, rounded to one decimal place.
- Features with blank or TBD estimate receive --bar-pct: 0%.

REQUESTER CLASS RULE:
- Read the "Requester" column for each feature row.
- "GRC" (case-insensitive) → class: req-grc  | label: GRC
- "GTM" (case-insensitive) → class: req-gtm  | label: GTM
- Anything else or blank   → class: req-other | label: Other

STATUS COLOR RULE — map Feature Status to dot color and text color:
- "In Progress"               → dot: #F0A500 | text: #b07800
- "In Design"                 → dot: #4A90D9 | text: #2a70b8
- "HLD"                       → dot: #4A90D9 | text: #2a70b8
- "Ready for Dev"             → dot: #4CAF50 | text: #2e7d32
- "To Do"                     → dot: #BBBBBB | text: #888888
- "Under Analysis"            → dot: #9B59B6 | text: #7d3c98
- "Waiting Analysis Approval" → dot: #E67E22 | text: #b85c10
- "Waiting Solution Approval" → dot: #E67E22 | text: #b85c10
- Blank                       → dot: #BBBBBB | text: #888888
