This feature is the result of a request from LT Project, however similar request exists from GR project since Tax officers currently can see highlighted required fields.  

*Functional Requirements*
Enhance the UI configuration to support a visual indicator for specific fields. Example use case:
 * Configurator may set the required items to be displayed with {color:#ff0000}*{color} ({color:#ff0000}red asterisk{color}).
 * Optional items displayed in normal text. No (Optional) label needed next to each field. 
 * Styling approach could be configurable to support different use cases (e.g., icons, color).

 * Existing functionality should be maintained as an option for *projects .* I.e., highlight Optional with text next to the fields.
