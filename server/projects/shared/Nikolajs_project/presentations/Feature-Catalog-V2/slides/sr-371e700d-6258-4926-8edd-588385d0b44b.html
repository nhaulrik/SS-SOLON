Key Epics & Features
Enable Financial Events Logging under Org Units and Claimants
Support logging of revenue type registrations, obligations, segments, and financial transactions under Organization Units and Claimants as party roles with balance visibility and party role overview pages.
GL Decisions Table - Multiline Rules and Additional Parameters
Introduce Valid From/To columns, multiline support with amount splitting rules, and extensible parameters for claimant, revenue type, and segment categorization to reduce GL decision table complexity.
GL Decisions Table - Extensibility and High Volume Support
Enable rule-driven splitting of GL decision tables across multiple XLSX sheets and files, support custom adapters for journal entry creation, and maintain performance for 200,000+ decision lines.
Revenue Type Administration for Party Roles
Allow administrators to define revenue types for use with Organization Units and Claimants, enabling flexible party role relationships and financial transaction categorization.
GL Decision Table Configuration and Documentation
Provide comprehensive documentation for configuration options, split strategies, reference implementations, and operational guidelines ensuring maintainability and admin usability.
Performance, Security, and Accessibility Standards
Maintain performance for large datasets and multi-sheet evaluation, ensure no exposure of sensitive configuration data, and keep configuration screens and documentation accessible.
