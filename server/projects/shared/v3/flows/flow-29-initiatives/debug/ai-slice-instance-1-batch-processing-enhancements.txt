Scope:
 # Update to newest version of Amplio
 ** Java Foundation 4.0 and Amplio Rho 2.0 are scheduled to be released in January 2026.
 ** The release will introduce new functionality for the batch engine, for example the possibility to start batch jobs directly from the UI. 
 ** The upgrade should be coordinated with the Amplioj4Solon team, as revenue-management-batch, Amplio Admin and Amplio Process Engine all need to follow the same version of Amplio.
 # Technical alignment between Amplio Process Engine, Amplio Batch and SOLON
 ** Because of the direct dependencies between the components, It is needed to get them technically aligned. This is both in regards to ensuring immediate support for new functionality but also ensuring maintainability of the projects within the SOLON product team.
 # Use foundation functionality for batch jobs
 ** The scope is both in regards to utilizing improved batch scheduler functionality introduced in future foundation releases, but also ensuring proper multitenancy support, as provided by the foundation.
