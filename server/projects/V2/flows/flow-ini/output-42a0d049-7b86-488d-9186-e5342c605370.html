TAXPAYER ACCOUNTING
As an administrator responsible for configuring the GL Decision Table (Accountant table), iI need to support the extensibility of the solution. I need to introduce additional parameters that control how GL accounts are split across multiple files or tables. This is required to support very high volumes of decision lines and complex business rules. I need to allow custom adapters to be used on how rules are created and selected to generate journal entries. These parameters will allow the system to efficiently handle hundreds of special cases, reduce the size and complexity of individual GL decision tables, and improve maintainability by splitting configurations into multiple, logically grouped records. A realistic number of GL decision lines with the assumption that legacy GLs will be removed from time to time, is 200.000 lines. Also includes support for tax account overview, tax account reference implementation, and configurable GL decision behavior.
