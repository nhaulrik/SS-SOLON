=== Strategic Case — Why Prioritise This Initiative ===

Reason 01 Title: Enable Intelligent Case Worker Assignment
Reason 01 Text: Capability to assign case workers based on availability, skills, and organisational unit membership ensures optimal resource allocation and improves case completion rates across revenue authority operations.

Reason 02 Title: Support Availability-Based Profiling
Reason 02 Text: Case worker profile configuration using availability criteria enables administrators to define matching rules that automatically identify suitable candidates, reducing manual assignment overhead and improving case outcomes.

Reason 03 Title: Enhance Case Effort Visibility
Reason 03 Text: Automatic effort calculation based on resource calendars and expected allocation provides transparency on case workload, enabling better capacity planning and resource management across case management teams.
