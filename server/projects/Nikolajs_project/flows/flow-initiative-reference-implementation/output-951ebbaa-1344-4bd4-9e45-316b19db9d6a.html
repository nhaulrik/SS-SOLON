Why It's Important
01
Self-Service Capability
Enables taxpayers to independently file returns and process payments through an intuitive digital interface, reducing dependency on support teams.
02
Operational Efficiency
Streamlines filing and payment workflows, reducing manual processing time and operational costs while improving accuracy and compliance.
03
Enhanced User Experience
Provides a modern, accessible platform for taxpayers to manage their filings and payments securely with real-time status tracking and notifications.
Key Epics & Features
Digital Filing Portal
Build user-friendly web interface enabling taxpayers to submit returns electronically with guided workflows, form validation, and real-time error detection.
Payment Processing
Implement secure payment gateway supporting multiple payment methods including credit cards, bank transfers, and digital wallets with PCI compliance.
Status Tracking
Develop real-time tracking system allowing taxpayers to monitor filing status, payment confirmation, and refund progress through personalized dashboard.
Document Management
Create secure document upload and storage system for supporting tax documents with version control, audit trails, and secure retrieval capabilities.
Authentication & Security
Implement multi-factor authentication, encryption, and role-based access controls to ensure secure taxpayer data protection and regulatory compliance.
Integration & APIs
Build RESTful APIs and integration points with backend tax processing systems, payment processors, and third-party services for seamless data flow.

Epics

Requester
Netcompany
Why It's Important
01
Tax Account Reference Implementation
Enable revenue officers to efficiently manage VAT debts and payments through reconciliation periods, providing a comprehensive demonstration of Solon Tax's capability to support VAT Tax Account requirements.
02
Compliance Warning System
Reduce taxpayer compliance errors by providing clear field labeling and proactive warnings about VIES Statement obligations for intracommunity transactions.
03
Demo Capability Showcase
Establish reference implementations vital for demonstrations and future implementation needs, showcasing Solon Tax's warning capabilities and tax account management features.
Key Epics & Features
VAT Tax Account Reference Implementation
Configure Revenue Types with tax account and reconciliation period support, manage reconciliation periods with opening/closing balances, enable financial transaction linking, and provide specialized UI for balance management and debt/payment processing.
Revenue Type Configuration
Create "VAT for Tax Account" (PERIODIC with FILING and PAYMENT actions) and "VAT for Tax Account Ad hoc" (Ad Hoc with PAYMENT action) revenue types with taxAccount and reconciliationPeriod flags enabled.
Reconciliation Period Management
Configure reconciliation period entities with period codes, start/end dates, and active/inactive status. Link to revenue type registrations with opening/closing balances and party informed indicators.
Balance Management & Processing
Support opening balance recording, automatic closing balance calculation, balance transfer between periods, manual adjustments with audit trail, debt recording, payment processing, cross-period allocation, and interest/penalty calculation.
VAT Return Form - Intracommunity Warning
Update field labels for intracommunity transactions in VAT Return Form and implement warning mechanism that displays compliance message when intracommunity fields contain non-zero values.
Field Label & Warning Implementation
Correct field labels for Intracommunity Sales, Intracommunity Sales without VAT, Intracommunity Purchases, and Intracommunity Purchases without VAT. Display VIES Statement submission deadline warning with amber styling and warning icon.
