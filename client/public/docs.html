🚀 Quick Start

Get up and running in 5 minutes. Follow these steps to transform your PPTX templates into data-driven presentations.

Prerequisites: Node.js (v18+), npm, git
1

Clone the Repository

Open your terminal and clone the project:

git clone <your-repo-url> cd SOLON
2

Install Dependencies

Install all dependencies from the root folder:

npm install

This installs both frontend and backend dependencies in one go.

3

Start the Application

Run a single command from the root folder:

npm run dev

This starts both the backend (port 3001) and frontend (port 5173) simultaneously.

4

Use the App

  1. Open http://localhost:5173 in your browser
  2. Upload a .pptx file
  3. Click on text elements to tag them as placeholders
  4. Toggle the AI switch for fields you want AI to generate
  5. Click Generate Recipe and copy the prompt
  6. Paste AI's JSON response and Preview
  7. Download your generated PPTX

What is SOLON Slide Studio?

SOLON Slide Studio is a web application that enables users to transform PowerPoint templates into dynamic, data-driven presentations. It solves the repetitive problem of manually updating slides for recurring presentations like weekly reports, sales decks, and status updates.

Core Idea: Tag elements in a PPTX template as placeholders, generate a recipe prompt for AI, paste AI-generated JSON, and automatically produce a finished presentation.

Problems Solved

🔄

Manual Repetition

No more copying and pasting the same slide structure for recurring presentations.

🤖

AI Integration Gap

Bridges the gap between online AI tools and local file generation.

💾

Template Reuse

Save patches (tag configurations) to disk for reuse across presentations.

👁️

Preview Before Download

Verify the final output before generating the file.

Walkthrough: Step by Step

1

Upload PPTX Template

Drag and drop your PowerPoint file (.pptx). The app parses the file and extracts all text elements with their positions, sizes, and styles.

↓
2

Tag Elements

Click on text elements in the slide preview to tag them as placeholders. Each tag becomes a variable that will be replaced with data later.

↓
3

Save Patch (Optional)

Give your patch a name and save it. Patches are stored in the /patches folder as JSON files for future reuse.

↓
4

Generate Recipe

Click "Generate Recipe" to create an AI prompt. This tells an online AI exactly what JSON structure you need.

{ "title": "...", "records": [{ "item": "..." }] }
↓
5

Copy Recipe → Paste JSON

Copy the recipe to AI, get JSON back, paste it into the app. Validation happens automatically as you type.

↓
6

Preview & Download

Review all generated slides, use arrow keys to navigate, then click "Download PPTX" to get your finished presentation.

How It Differs from Browser-Only AI

Aspect Browser-Only AI SOLON Slide Studio
File Generation Cannot create PPTX files Generates downloadable PPTX
Template Handling Copy-paste workflow Parses PPTX, preserves formatting
Data Binding Manual replacement Automatic placeholder replacement
Preview No visual preview Real-time slide preview
Reusability No templates/patches Patches persist for reuse
The Bridge: SOLON Slide Studio acts as a bridge between AI (which generates JSON) and PowerPoint (which needs the actual file).

Technical Overview

Architecture

  • Frontend: React + Vite (port 5173)
  • Backend: Node.js + Express (port 3001)
  • PPTX Parsing: adm-zip library
  • Persistence: JSON files in /patches

Placeholder Convention

Elements are tagged using Mustache-style syntax:

{{product_name}}
